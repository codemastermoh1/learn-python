اختبار بايثون

اختبر مهاراتك في بايثون!

اختبار مستوى مبتدئ

ما نوع البيانات الذي يُستخدم لتخزين النصوص في بايثون؟

  • int
  • float
  • str
  • list

أي من الخيارات التالية يستخدم لتعليق سطر واحد في بايثون؟

  • /* */
  • //
  • #

أي من الكودات التالية يقوم بتحويل النص "123" إلى عدد صحيح في بايثون؟

  • int('123')
  • str('123')
  • float('123')
  • convert('123')

اختبار مستوى متوسط

ما هي النتيجة عند تنفيذ الكود التالي؟ 

x = [1, 2, 3]
x.append([4, 5])
print(len(x))

  • 4
  • 5
  • 6
  • 3

أي من الخيارات التالية يُنتج عددًا عشوائيًا بين 1 و10 (شاملًا)؟

  • random.randint(1, 10)
  • random.random(1, 10)
  • random.range(1, 10)
  • random.uniform(1, 10)

ما هي النتيجة التي ستظهر عند تنفيذ الكود التالي؟ 

x = [1, 2, 3, 4, 5]
print(x[-1])

  • 1
  • 5
  • 4
  • خطأ

أي من الخيارات التالية يعبر عن طريقة تعريف دالة في بايثون؟

  • function myFunc() {}
  • def myFunc():
  • myFunc() => {}
  • function: myFunc

اختبار مستوى محترف

ما هي نتيجة الكود التالي؟ 

x = [1, 2, 3]
y = x
y[0] = 0
print(x)

  • [1, 2, 3]
  • [0, 2, 3]
  • [1, 0, 3]
  • خطأ في الكود

ما هي النتيجة عند تنفيذ الكود التالي؟ 

def my_func(a, b=[]):
    b.append(a)
    return b

print(my_func(1))
print(my_func(2))

  • [1], [2]
  • [1], [1, 2]
  • Error
  • خطأ في الكود

ما هي النتيجة عند تنفيذ الكود التالي؟ 

x = {1, 2, 3}
y = x
x.add(4)
print(y)

  • {1, 2, 3, 4}
  • {1, 2, 3}
  • Error
  • خطأ في الكود
