أساسيات Python

دليل شامل للمبتدئين لفهم الأساسيات المهمة في لغة البرمجة Python.

طباعة (print)

في بايثون، يتم استخدام دالة print() لطباعة النصوص أو القيم إلى الشاشة. يمكن طباعة نصوص ثابتة أو متغيرات أو نواتج عمليات معينة باستخدام هذه الدالة.

print("مرحباً بالعالم!")
        
النتيجة  مرحباً بالعالم!

المتغيرات (Variables)

المتغيرات هي مكان لتخزين البيانات. في Python، يمكننا تخزين أي نوع من البيانات في المتغيرات.

name = "Mohamed"  # نص
age = 14             # عدد صحيح
height = 1.75        # عدد عشري
النتيجة  سيخزنها في هزة المتغيرات ويمكن طباعتها 
            print(name) وسيطبع Mohamed

أنواع البيانات (Data Types)

Python يدعم أنواع بيانات متنوعة مثل النصوص (Strings)، الأعداد الصحيحة (Integers)، الأعداد العشرية (Floats)، والقوائم (Lists).

name = "Mohamed"   # نص
age = 14             # عدد صحيح
height = 1.75        # عدد عشري
list=[1,2,3,4]       # والقوائم

العمليات الحسابية (Arithmetic Operations)

يمكنك استخدام العمليات الحسابية الأساسية مثل الجمع (+)، الطرح (-)، الضرب (*)، والقسمة (/).

num1 = 10
num2 = 5
sum = num1 + num2
print("المجموع:", sum)

الشروط (If Statements)

الشروط تساعدنا في اتخاذ القرارات بناءً على القيم. باستخدام جملة if، يمكنك تنفيذ كود معين إذا تحقق شرط معين.

age = 14
if age >= 18:
    print("أنت بالغ.")
else:
    print("أنت قاصر.")

الحلقات (Loops)

الحلقات تتيح لك تنفيذ نفس الكود عدة مرات. في Python، يمكنك استخدام `for` و `while` لتكرار التعليمات البرمجية.

for i in range(5):
    print(i)

أساسيات أخرى

  • تعريف الدوال (Functions): الدوال هي عبارة عن كود مكتوب لتنفيذ مهمة معينة ويمكن استدعائها عدة مرات.
    • def greet(name):
    return "Hello " + name
  • القوائم (Lists): القوائم هي مجموعة من العناصر التي يمكن أن تكون من أي نوع بيانات ويمكن تعديلها.
    • fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # apple
  • التكرار باستخدام الدوال (Recursion): هو استدعاء الدالة لنفسها لتنفيذ عمليات متكررة.
    • def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)
  • المجموعات (Sets): هي مجموعة من العناصر غير المكررة.
    • my_set = {1, 2, 3, 4}
my_set.add(5)
  • القواميس (Dictionaries): هي مجموعة من الأزواج من المفاتيح والقيم.
    • person = {"name": "Mohamed", "age": 14}
print(person["name"])  # Mohamed
  • التعامل مع الملفات (File Handling): يمكنك قراءة وكتابة البيانات في الملفات باستخدام Python.
    • with open("file.txt", "w") as file:
    file.write("Hello, world!")
  • التعامل مع الأخطاء (Exception Handling): يمكن التعامل مع الأخطاء باستخدام `try-except`.
    • try:
    print(1 / 0)
except ZeroDivisionError:
    print("لا يمكن القسمة على صفر.")
  • الواردات (Imports): يمكنك استيراد مكتبات خارجية لاستخدامها في برنامجك.
    • import math
print(math.sqrt(16))  # 4.0
  • المكتبات القياسية (Standard Libraries): تحتوي Python على العديد من المكتبات الجاهزة التي يمكنك استخدامها مثل `math` و `random`.
  • التخصيص (Decorators): هي دوال تُستخدم لتعديل سلوك الدوال الأخرى.
    • def decorator(func):
    def wrapper():
        print("قبل الدالة")
        func()
        print("بعد الدالة")
    return wrapper

@decorator
def say_hello():
    print("مرحبًا")
  • الـ lambdas: هي دوال صغيرة يمكن استخدامها في مكانها مباشرة.
    • add = lambda x, y: x + y
print(add(2, 3))  # 5
