مكتبة OpenCV

تعريف: مكتبة OpenCV تُستخدم لمعالجة الصور والفيديوهات. تُعتبر من أقوى المكتبات المستخدمة في مجال الرؤية الحاسوبية (Computer Vision).

طريقة التثبيت:

pip install opencv-python

أمثلة:


# استيراد مكتبة OpenCV
import cv2

# قراءة صورة
image = cv2.imread('example.jpg')

# تحويل الصورة إلى التدرج الرمادي
gray_image = cv2.cvtColor(image, cv2.COLOR_BGR2GRAY)

# عرض الصورة
cv2.imshow('صورة رمادية', gray_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

مكتبة Pillow

تعريف: مكتبة Pillow تُستخدم لمعالجة الصور. تُعتبر خيارًا بسيطًا لتحرير الصور مقارنة بـ OpenCV.

طريقة التثبيت:

pip install pillow

أمثلة:


# استيراد مكتبة Pillow
from PIL import Image

# فتح صورة
image = Image.open('example.jpg')

# تغيير حجم الصورة
resized_image = image.resize((100, 100))

# حفظ الصورة الجديدة
resized_image.save('resized_example.jpg')

مكتبة PyTorch

تعريف: مكتبة PyTorch تُستخدم لتطوير نماذج الذكاء الاصطناعي والتعلم العميق. تُعتبر منافسًا قويًا لـ TensorFlow.

طريقة التثبيت:

pip install torch

أمثلة:


# استيراد مكتبة PyTorch
import torch

# إنشاء مصفوفة بسيطة
x = torch.tensor([1, 2, 3])

# إجراء عمليات رياضية
print(x + 2)  # إضافة 2 لكل عنصر

مكتبة Seaborn

تعريف: مكتبة Seaborn تُستخدم لإنشاء رسوم بيانية إحصائية بسهولة، وهي مبنية على مكتبة Matplotlib.

طريقة التثبيت:

pip install seaborn

أمثلة:


# استيراد المكتبة
import seaborn as sns
import matplotlib.pyplot as plt

# إنشاء بيانات
data = [10, 20, 20, 40, 50, 50, 50]

# رسم مخطط
sns.histplot(data, bins=5)
plt.show()

مكتبة PyGame

تعريف: مكتبة PyGame تُستخدم لتطوير الألعاب ثنائية الأبعاد بسهولة.

طريقة التثبيت:

pip install pygame

أمثلة:


# استيراد المكتبة
import pygame

# تهيئة PyGame
pygame.init()

# إنشاء نافذة
screen = pygame.display.set_mode((400, 300))

# اللون الأحمر
red = (255, 0, 0)

# رسم مستطيل
pygame.draw.rect(screen, red, (50, 50, 100, 50))

# تحديث الشاشة
pygame.display.flip()

# انتظار 5 ثواني قبل الإغلاق
pygame.time.wait(5000)
